Лабораторная работа №6
«Реализация REST API»

1.Перед запуском сервера требуется запустить базу данных. 
	В командной строке перейти в директорию базы данных MongoDB.
	В текущей директории перейти в папку «Server/4.2/bin».
	В командной строке выполнить команду «mongod».
2.Запуск сервера.
	Для запуска сервера необходимо перейти в папку с проектом.
	Выполнить команду «node index».
	Далее перейти по ссылке «localhost:8888/api»


Лабораторная работа №7
«Frontend на ReactJS»
#Запуск текущей лабораторной работы по большей части 
будет аналогичен запуску лабораторной работы №6#

1.Перед запуском сервера требуется запустить базу данных. 
	В командной строке перейти в директорию базы данных MongoDB.
	В текущей директории перейти в папку «Server/4.2/bin».
	В командной строке выполнить команду «mongod».
2.Запуск сервера.
	Для запуска сервера необходимо перейти в папку с проектом.
	Выполнить команду «node index».
	Запустить приложение React.
	Перейти в папку с приложением React.
	Выполнить команду «npm start».
	В браузере должно открыться приложение.
